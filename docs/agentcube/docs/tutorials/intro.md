# Tutorial Overview

Welcome to the AgentCube tutorials! These guides are designed to take you from a beginner to an expert in managing AI Agent workloads on Kubernetes.

## What You'll Learn

In this section, we cover:

1. [Your First Agent](./first-agent.md): A deep dive into the `AgentRuntime` custom resource and how to deploy a basic agent.
2. [Using the Python SDK](./python-sdk.md): How to programmatically interact with isolated sandboxes using our official SDK.
3. [Real-world Example: PCAP Analyzer](./pcap-analyzer.md): A look at a sophisticated multi-agent system built on top of AgentCube.

## Prerequisites

Before starting these tutorials, please ensure you have followed the **[Getting Started](../getting-started.md)** guide to install AgentCube on your cluster.

:::tip
Most of these tutorials can be run on a local [Kind](https://kind.sigs.k8s.io/) or [Minikube](https://minikube.sigs.k8s.io/docs/start/) cluster.
:::
